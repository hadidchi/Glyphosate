# Glyphosate
This is the code used to analyze the association between county-level glyphosate and suicide mortality in the United States from 1992-2019. 
Download the Jupyter notebook for the code. 
For the data, please see https://drive.google.com/file/d/1gxNXeZiAUIRfR2053ENYFYy0WwIJrM20/view?usp=sharing.
